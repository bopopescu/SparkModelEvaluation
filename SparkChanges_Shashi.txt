Spark Changes:

1. Created a trait METrigger, which is implemented by SparkContext to trigger dynamic model evaluation.
spark-1.4.0\core\src\main\scala\org\apache\spark\METrigger.scala

2. SparkContext.scala: SparkContext object also implements and overrides the setMETrigger and getMETrigger functions of trait METrigger to trigger dynamic model evaluation.
spark-1.4.0\core\src\main\scala\org\apache\spark\SparkContext.scala

3. SVM3.scala : SVM3 is the binary svm class implemented with dynamic model evaluation through checkpointing
spark-1.4.0\mllib\src\main\scala\org\apache\spark\mllib\classification\SVM3.scala

4. GradientDescent3.scala : GradientDescent3 also implements dynamic model evaluation and checkpointing, which is currently used by SVM3
spark-1.4.0\mllib\src\main\scala\org\apache\spark\mllib\optimization\GradientDescent3.scala

5. Client.scala : Extended Client.scala to read RMRequestState from the ApplicationReport generated by YARN. RMRequestState sets the METrigger in the SparkContext to true.
spark-1.4.0\yarn\src\main\scala\org\apache\spark\deploy\yarn\Client.scala

6. yarnClientSchedulerBackend.scala : Edited the waitForApplication() function to accomodate RMRequestState, while running the monitorApplication() function.
spark-1.4.0\yarn\src\main\scala\org\apache\spark\scheduler\cluster\YarnClientSchedulerBackend.scala
